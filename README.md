# Time-Stamped Sensor Data Logger using LPC2148
## Overview
This project implements a real-time temperature monitoring and data logging system using the LPC2148 ARM7 microcontroller. The system records temperature data along with accurate date and time using the on-chip RTC and transmits the information to a serial terminal.

## Features
- Real-time temperature monitoring using LM35 sensor
- RTC based time stamping
- 16x2 LCD display output
- UART serial communication logging
- Over-temperature alert system
- User editable time and setpoint using keypad
- Embedded C implementation

## Hardware Used
- LPC2148 ARM7 Microcontroller
- LM35 Temperature Sensor
- 16x2 LCD
- Matrix Keypad
- MAX232 IC
- LED/Buzzer

## Software Used
- Keil uVision
- Flash Magic
- Embedded C

## Applications
- Industrial Monitoring
- Environmental Data Logging
- Laboratory Systems
- Safety Monitoring
